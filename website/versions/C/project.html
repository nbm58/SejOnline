Project Overview



Sej online is a network connected multiplayer game of chance. While revisiting an old book he had read, our client was curious if a game from the book was possible to create in real life. Sej online aims to bring this vision to full fruition, Full release Spring 2023!

Requirements



Sej online must implement the game rules from the book
Sej must be capable of a network connected two player mode.
The UI for sej must be Simple, yet attractive
User accounts must have persistent score keeping for opponents

Solution



The team will begin with a small prototype build. Once this is
successful, we will rework and improve the game, adding new features
and improving upon older ones.
